# Project 02 – Zombie 

Last update on September 23, 2022

Arcade topdown zombie shooter 

## Implementation
Player aims with the mouse and moves with WASD controls corresponding to up, left, down, and right
Zombies spawn in periodically (faster as the game progresses) and will chase the player

Uses art assets from Kenney.nl
